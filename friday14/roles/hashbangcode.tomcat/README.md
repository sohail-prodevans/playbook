# Ansible Role: Tomcat

An Ansible role that installs [Tomcat](http://tomcat.apache.org/) on RHEL/CentOS and Debian/Ubuntu.


## Requirements

None.


## Role Variables

None.


## Dependencies

None.


## Example Playbook

```
- hosts: servers
  roles:
     - { role: hashbangcode.tomcat }
```
